## 安装

### CDN {page=#/install}

由于目前组织正在发展，没钱单独购置CDN，因而 bk-vue-cec 不支持CDN引入 

### NPM 安装 {page=#/install}

组件库已发布至 [npm](https://www.npmjs.com/package/bk-vue-cec)，只需要在项目根目录下执行以下命令便可以安装 `bk-vue-cec`。

```bash
$ npm i bk-vue-cec --save
```

我们更加推荐通过 `yarn` 安装 `bk-vue-cec`，并使用 `webpack` 作为构建工具，命令如下：

```bash
$ npm install -g yarn
$ yarn add bk-vue-cec
```
